# Sistema Escolar – POO
Projeto desenvolvido com o objetivo de praticar os conceitos de **Programação Orientada a Objetos (POO)**, utilizando **classes, herança e polimorfismo**, por meio de um sistema de gestão escolar com alunos e professores.

##  Descrição doProjeto
O sistema representa membros de uma escola por meio de uma classe base chamada `SchoolMember`, da qual derivam as classes `Student` e `Teacher`.  
Cada tipo de membro possui comportamentos e responsabilidades específicas, respeitando regras próximas da realidade escolar.
